退换货政策

退换货：  商品自实际收货之日起8天内，可为客户提供退换货服务；特殊商品除外。 进货商需自行以快递形式寄回商品（请勿选择到付方式），退回地址请以系统后台地址为准。如进货商需要换货，纳车品可为进货商提供退换货服务，换货时仅可更换为同款商品。

符合以下条件，可以要求换货： 

1. 在收到货物时当面在送货员面前拆包检查，发现货物有质量问题的。

2. 实际收到货物与网站上描述的有不一致的。

换货流程：  当面要求送货人员退回货物，然后与我们联系。纳车品会为您重新发货，货物到达时间顺延。

符合以下条件，可以要求退货： 

1. 客户收到货物后三天之内，发现商品有明显缺陷的。

2. 货物经过一次换货但仍然存在质量问题的。

在以下情况纳车品有权拒绝客户的退换货要求： 

1. 货物出现破损，但没有在收货时当场要求送货人员换货的。

2. 超过退换货期限的退换货要求。

3. 商品留有洗涤的痕迹。

4. 客户发货单据丢失或者不全。

5. 产品并非纳车品提供。

6. 货物本身不存在质量问题的。

7. 退换货之前未与纳车品取得联系私自将货物退货的。

8. 不可抗力导致的损坏。

9. 根据每台显示器像素和调比度不同而产生的商品图片与实物的视觉差异并非商品质量问题。

换货/退货邮费：  若因颜色款式发错问题导致的换货，来回邮费由纳车品承担；如果因非质量问题退货，则纳车品不承担运费。
