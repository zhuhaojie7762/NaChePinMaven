公司介绍

天台纳车品网络科技有限公司隶属于金恒德集团有限公司，是一家专注于中国汽车后市场,为汽车用品生产和销售企业提供一站式电子商务解决方案的互联网公司，纳车品公司目前已经在中国汽车用品集散地“天台县”建立起以“服务于汽车用品厂家和网店卖家的供销一 体平台，加速中国汽车用品电子商务流通发展的电子商务孵化产业园，发扬大众创业、万众创新并为广大青年大学生提供创业孵化区和天台汽车用品产销集散地物流平台集聚和解决方案”四方为一体的生态平台，通过互联网平台和电子商务产业园，打通天台汽车用品集散地的供销关系，发挥天台汽车用品集散地的产销优势，为致力于打造天台质造的全国化乃至国际化品牌而不断努力。
