退换货办理流程

申请提交：  网上办理：登录进货商后台所在订单下直接申请。

退换货商品退回方式说明： 

1. 纳车品确认您的需求后，请尽快将商品退回至指定地址。

2. 退回商品时，请将商品、发货单、发票一起退回。

3. 包裹寄到后，我们会对退回的商品进行质检，再根据需求进行退换货操作。

4. 纳车品不接收到付件。

5. 温馨提示：纳车品不接受任何形式的到付和平邮，因此退换货商品请务必以快递或者物流方式寄回，否则您的包裹将拒收退还给您。
